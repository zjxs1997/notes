## JavaScript xjb记

- js的字符串里有个反引号，\`，这个可以用来写python中三个引号的换行字符串，也可以写类似于f-string的东西。

```javascript
var s = `dsfsd
sdfsd
sdf`;
console.log(s);

var name = 'tom';
var hello =`hello ${name}`;
console.log(hello);
```
